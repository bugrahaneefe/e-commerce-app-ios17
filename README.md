## Database

- Database is implemented by SwiftData

## Description

- This app is e-commerce platform app which is a marketplace template for cosmetics.
- Database need to consist of two tables:
  - ProductModels
    - Consist of details of products. 
  - ProductCategoryMdeols
    - Consist of every product categories included in database.
